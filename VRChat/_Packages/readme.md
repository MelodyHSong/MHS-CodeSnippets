# ☆ Compiled Unity Packages ☆

> "Ready to drag-and-drop, no assembly required!"

This directory contains finalized **.unitypackage** files. These packages bundle specific sets of scripts, prefabs, and editor tools for easy, clean installation into your Unity VRChat projects.

---

## ☆ Installation Instructions

To install any asset found here:

1.  **Download:** Download the desired **`.unitypackage`** file directly from this folder.
2.  **Open Unity:** Ensure your Unity VRChat project is open.
3.  **Import:** Drag the downloaded `.unitypackage` file directly into your Unity **Project** window, OR double-click the file on your computer.
4.  **Confirm:** In the import dialogue that appears, click **`Import`** to add all the contents to your project's `Assets` folder.
