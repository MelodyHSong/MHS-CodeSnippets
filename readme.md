# ☆ MHS-CodeSnippets ☆

> "My digital cookbook for reusable logic."

Welcome to **MHS-CodeSnippets**! This repository serves as a personal library for the code blocks, algorithms, and logic patterns that I find myself rewriting constantly. 

Whether it's a game mechanic for a specific engine or a utility class for a software application, if it's useful, it lives here.

### Languages

* **Python** (Scripting & Data Analysis)
* **U#** (VRChat UdonSharp)

### Engines & Environments
* **Unity**
