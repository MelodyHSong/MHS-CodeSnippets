# ☆ Internet Speed Tester ☆

> "Faster than a PT Cruiser, but hopefully more reliable."

A simple Python utility to measure your current network performance. It provides real-time data on your ping (latency), download speed, and upload speed using the global Speedtest.net server network.

## ☆ Installation & Prerequisites

This script requires Python 3.x and the `speedtest-cli` library.

### Quick Install

```powershell
python -m pip install speedtest-cli
``` 

## ☆ Usage
Run the script from your terminal or IDE. It will automatically locate the closest test server to ensure the most accurate results for your current location.

### Example Usage
```PowerShell

python NetSpeed.py
```

*Note: For the most accurate results, ensure no heavy background downloads or streaming services are active while the test is running.*
