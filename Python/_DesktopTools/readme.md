# ☆ Python Desktop Tools ☆

> "Automating the mundane, one script at a time."

Welcome to the **Python Desktop Tools** subdirectory! This collection is dedicated to standalone Python utilities designed to enhance desktop productivity, manage system resources, and streamline local workflows.

## ☆ Installation & Prerequisites

To use the tools in this directory, ensure you have Python installed and the necessary libraries.

### Quick Install
```bash
python -m pip install tqdm
```

## ☆ Usage

### **Standard Terminal** 

- Navigate to this directory.

- Run the desired script, for example:

```Bash

python custom_drive_analyzer.py

```

### **VS Code (Alternate)**
- Open the script in VS Code.

- Navigate to the top menu: Run > Start Debugging (or press F5).

- Alternatively, right-click the editor and select Run Python File in Dedicated Terminal.
